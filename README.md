# Cloud Tu Nube — Sitio Web Corporativo

Sitio web corporativo desarrollado para **Cloud Tu Nube**, empresa chilena de Auditoría y Gestión. El proyecto fue construido con HTML, CSS y Bootstrap a partir de una plantilla personalizada.

## Vista previa

Disponible en: [Tyler0125.github.io/Paginaweb.github.io](https://Tyler0125.github.io/Paginaweb.github.io)

## Tecnologías

- HTML5
- CSS3
- Bootstrap 4
- jQuery
- Font Awesome

## Páginas

| Página | Descripción |
|---|---|
| `index.html` | Página principal: Nosotros, Misión/Visión, Servicios y Contacto |
| `about.html` | Descripción extendida de la empresa |
| `clients.html` | Detalle de los 6 servicios ofrecidos |
| `contact.html` | Formulario de contacto |
| `ourwork.html` | Galería de trabajos realizados |

## Servicios que ofrece Cloud Tu Nube

- **Contabilidad** — Análisis de movimientos económicos y financieros
- **Tributación** — Declaración de Impuesto a la Renta y régimen tributario
- **Recursos Humanos** — Contratos, finiquitos, planillas y libros
- **Finanzas** — Obtención de recursos, ahorro e inversión
- **Auditoría** — Evaluación de herramientas para minimizar riesgos
- **Capacitación** — Educación financiera y tributaria
